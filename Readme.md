# 📝 Blog Platform

An interactive blog platform that allows users to create, edit, and manage blog posts with a user-friendly interface. Built with **React**, **Node.js**, and **MongoDB** for seamless user experiences.

## 🚀 Features

- **Authentication**: Secure login and user-specific content access.
- **Interactive Blog Management**:  
  - Create, edit, and delete your blog posts.  
  - Tag-based categorization with a sleek hashtag editor.  
- **Dynamic UI**: Fully responsive design with Tailwind CSS.  
- **Backend**: Robust RESTful API powered by Node.js and MongoDB.  

---

## 🛠️ Tech Stack

**Frontend**: React, Tailwind CSS, React Router  
**Backend**: Node.js, Express.js, MongoDB  
**Authentication**: JWT-based secure login  
**API Requests**: Axios for seamless communication between client and server

## 🏗️ Installation & Setup

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/blog-platform.git
   cd blog-platform
   ```

2. Install dependencies for both frontend and backend:
   ```bash
   # Frontend
   cd client
   npm install

   # Backend
   cd ../server
   npm install
   ```

3. Create a `.env` file in the backend folder with the following:
   ```env
   MONGO_URI=your_mongo_db_connection_string
   JWT_SECRET=your_key
   PORT=5000
   ```

4. Run the development servers:
   ```bash
   # Frontend
   cd client
   npm i && npm run dev

   # Backend
   #cd into server file
   npm i 
   npm run dev
   ```

5. Open your browser and visit: `http://localhost:5173`.
